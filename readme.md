# 命令行压缩工具使用说明

## 文件依赖
- library.zip
- python27.dll
- pyzip.exe

如果以上文件缺失, 软件将无法正常运行


## 使用方法
1. 以管理员启动cmd命令行(如非管理员权限,将无法对C盘进行读操作)
2. 执行 pyzip.exe  [源文件路径] [压缩包名(绝对路径)]

> 例如： pyzip  "d:/test_folder/" "d://test.zpi"

如果仅输入第一参数: 源文件路径, 软件将在当前路径下生成  “default.zip” 压缩包
